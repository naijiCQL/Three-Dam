# three-dam

## 项目介绍

```
使用three.js+vue2建造湖北中小水库三维模型，其中包括河床、动态水体、坝体、渗压计以及渗压计数值显示。
```

### Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
