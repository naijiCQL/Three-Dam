# Three-Dam

## 项目介绍

 1、使用three.js+vue2建造**湖北中小水库三维模型**，其中包括河床、动态水体、坝体、渗压计以及渗压计数值显示。   

 2、三维物体构建代码位于**create3DObjects.js**中，数据渲染代码位于**index.js**中。

## 项目启动

### 依赖安装

> npm install

### 启动项目

> npm run serve
